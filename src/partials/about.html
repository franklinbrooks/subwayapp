This app displays some information about the New York City Subway system in order to demonstrate some properties of Angular 2, a front-end web framework developed by Google.


*


Angular 2 was developed to display data client-side, via two-way binding, in a way that minimizes server calls. This reduces web traffic and improves user experience.


*


All information is pulled from the server at page load. Subsequent user navigation merely re-orders this information in the browser.


*


Learn more at the Angular2 website.
